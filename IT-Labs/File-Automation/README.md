\# File Renamer \& Backup Script (Python)



\*\*Description:\*\*  

A simple Python automation script that scans the current directory for `.txt` files and automatically renames them by adding a `\_backup` suffix. Useful for quick backups, file organization, and basic automation tasks.



\*\*Technologies:\*\* Python, OS automation, file management



\*\*How to run:\*\*

```bash

python file\_renamer.py
